Proteomic Data Analysis (PaDuA)
===============================

PaDuA is a Python package to simplify the processing and analysis of quantified proteomics data. Currently it supports
processing and analysis of MaxQuant outputs, providing many of the features available in the GUI analysis tool Perseus.
By scripting these processing and analysis steps you can get to your results more quickle and reproducibly.
